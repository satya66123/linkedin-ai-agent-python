# ⚠️ Problems & Solutions

## ❌ Ollama Timeout

**Issue:** Request taking too long
**Solution:**

* Use phi3 model
* Increase timeout
* Reduce prompt size

---

## ❌ MySQL Connection Error

**Issue:** Access denied
**Solution:**

* Check username/password
* Ensure MySQL is running

---

## ❌ Flask Not Running

**Issue:** App not starting
**Solution:**

* Check virtual environment
* Install dependencies

---

## ❌ Empty AI Response

**Issue:** No output
**Solution:**

* Check Ollama running
* Validate prompt

---

## ❌ Postman Body Missing

**Issue:** No JSON body
**Solution:**

* Select POST
* Body → raw → JSON

---

## ❌ Slow Performance

**Issue:** Takes minutes
**Solution:**

* Use phi3
* Limit tokens
* Reduce prompt size
