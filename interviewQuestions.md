# 🎯 Interview Questions & Answers

## Q1: What is Ollama?

Ollama is a tool that allows running large language models locally without cloud APIs.

---

## Q2: Why use local LLM instead of OpenAI?

* No cost
* Privacy
* Offline capability

---

## Q3: Explain your project architecture

* DTO → input validation
* Service → AI logic
* Repository → DB layer
* Routes → API endpoints

---

## Q4: How did you optimize performance?

* Used smaller models (phi3)
* Limited token generation
* Reduced prompt size

---

## Q5: What challenges did you face?

* Ollama latency
* Model size issues
* DB connectivity

---

## Q6: How can this project be scaled?

* Add caching
* Use async processing
* Deploy with GPU support

---

## Q7: Future improvements?

* User authentication
* React frontend
* AI analytics

---

## Q8: What is REST API?

An API that follows HTTP methods like GET, POST for communication between client and server.
